**1. Clone the repository:**

**4. Run the project in development or production mode:**

*For development*
```bash

```

*For production*
```bash

```

**5. Run tests:**

*Unit tests*
```bash
npm run test:unit
```
*Integration tests*
```bash
npm run test:integration
```
